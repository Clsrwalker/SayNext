本地模式下，你本机需要跑：

cd D:\SayNext
$env:Path += ";C:\Users\Admin\.bun\bin"
bun run dev
Ollama 如果已经在运行，不需要再 ollama serve。

frpc 已经在后台跑了；需要重启时：

Start-Process "$env:USERPROFILE\.saynext\frp\frpc.exe" `
  -ArgumentList "-c `"$env:USERPROFILE\.saynext\frp\frpc.toml`"" `
  -WindowStyle Hidden

开 3 个 PowerShell 窗口分别跑。

窗口 1：SayNext

cd D:\SayNext
$env:Path += ";C:\Users\Admin\.bun\bin"
bun run dev
窗口 2：Ollama

$env:Path += ";$env:LOCALAPPDATA\Programs\Ollama"
ollama serve
如果提示 Ollama 已经在运行，就不用管。

窗口 3：frpc

Start-Process "$env:USERPROFILE\.saynext\frp\frpc.exe" `
  -ArgumentList "-c `"$env:USERPROFILE\.saynext\frp\frpc.toml`"" `
  -WindowStyle Hidden
